## Tutorial

Run a benchmark to compare different providers:

```bash
trainloop benchmark
```

## Howâ€‘to

Benchmarks use suites defined in `trainloop/eval/`. Ensure each suite contains calls for the providers you want to compare.

## Reference

Generates benchmark result files under `trainloop/data/benchmarks/{timestamp}/` for each suite. The format is described in the [Benchmark Data Formats](../docs/reference/data-formats/benchmark-data-formats.md).

## Explanation

Benchmarking runs the same prompts against multiple LLM providers and records the results. You can then visualize the comparisons in the Studio UI.
